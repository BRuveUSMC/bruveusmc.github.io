How does this tool work?
The injector analyzes your uploaded file for patterns in TeaVM's compiled JavaScript code. It replaces functions with proxies to original code, moving them into ModAPI.hooks.methods. Similar processing applies to static properties and constructors, with hooks into $rt_metadata for auxiliary information.
